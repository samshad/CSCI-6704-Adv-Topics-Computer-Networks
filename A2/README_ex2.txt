# Exercise 2

This Python program performs bit stuffing and unstuffing operations on a given hex string. It converts the hex string to binary, stuffs the bits, unstuff them, and converts the binary back to the original hex string.

## Requirements
You need Python installed to run this program. You can download Python from [here](https://www.python.org/downloads/).


### Running the script

You can run the script using the following command:
```bash
python exercise2.py
```
