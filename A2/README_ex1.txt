# Exercise 1

This Python script simulates three digital encoding schemes: Unipolar, NRZ, and Manchester, based on an input binary string. The waveforms are displayed using Matplotlib.

## Requirements
To run the script, you need to have Python installed along with the Matplotlib and NumPy libraries.

### Installation of Matplotlib and NumPy
You can install Matplotlib and NumPy using pip:
```bash
pip install matplotlib numpy
```

### Running the script

You can run the script using the following command:
```bash
python exercise1.py
```
